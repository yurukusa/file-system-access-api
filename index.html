API Reference Code
// Open a file
const [handle] = await window.showOpenFilePicker({
  types: [{
    description: 'Text Files',
    accept: { 'text/plain': ['.txt', '.md'] }
  }],
  multiple: false
});

// Read contents
const file = await handle.getFile();
const text = await file.text();

// Write to file (requires permission)
const writable = await handle.createWritable();
await writable.write('New content');
await writable.close();

// Save as new file
const newHandle = await window.showSaveFilePicker({
  suggestedName: 'output.txt'
});

// Browse directory
const dirHandle = await window.showDirectoryPicker();
for await (const [name, entry] of dirHandle) {
  console.log(name, entry.kind); // 'file' or 'directory'
}

Details

About

The File System Access API allows web apps to read and write files on the user's local file system. All operations require user interaction (picker dialog) — no silent access.

API Methods

MethodReturns
showOpenFilePicker()FileSystemFileHandle[]
showSaveFilePicker()FileSystemFileHandle
showDirectoryPicker()FileSystemDirectoryHandle
handle.getFile()File
handle.createWritable()FileSystemWritableFileStream
handle.queryPermission()'granted'|'denied'|'prompt'

Handle Types

TypeProperties
FileSystemFileHandlekind:'file', name, getFile(), createWritable()
FileSystemDirectoryHandlekind:'directory', name, getFileHandle(), getDirectoryHandle(), entries()

Security Model

• Requires user gesture (click/keyboard)
• Picker dialog always shown (no silent access)
• Write permission asked separately
• HTTPS or localhost only
• No access outside picked scope
• Permission resets on page reload
